libpg_query gives you direct access to the PostgreSQL query parser from any application. It uses the actual PostgreSQL server source to produce parse trees, token streams, and normalized query forms — without running a database.

## Quick example

```c simple.c
#include <pg_query.h>
#include <stdio.h>

int main() {
  PgQueryParseResult result = pg_query_parse("SELECT 1");

  printf("%s\n", result.parse_tree);

  pg_query_free_parse_result(result);
  return 0;
}
```

Compile and link against the library:

```bash
cc -Ilibpg_query -Llibpg_query example.c -lpg_query
```

<Note>
  libpg_query is primarily a base library. If you are working in Ruby, Go, Python, Rust, or Node.js, consider using one of the [higher-level wrappers](/reference/language-wrappers) that wrap this library.
</Note>
