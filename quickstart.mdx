title: "Quickstart"
description: "Parse your first SQL query with libpg_query in minutes."
---

This guide walks you through building libpg_query and parsing a SQL query from a C program.

<Steps>
  <Step title="Clone and build the library">
    ```bash
    git clone -b 17-latest git://github.com/pganalyze/libpg_query
    cd libpg_query
    make
    ```

    <Note>The build compiles parts of PostgreSQL and may take up to 3 minutes.</Note>
  </Step>

  <Step title="Write a C program">
    Create `example.c` in the parent directory (next to the `libpg_query/` folder):

    ```c example.c
    #include <pg_query.h>
    #include <stdio.h>
    #include <stdlib.h>

    int main() {
      PgQueryParseResult result;

      result = pg_query_parse("SELECT 1");

      if (result.error) {
        printf("error: %s at %d\n", result.error->message, result.error->cursorpos);
      } else {
        printf("%s\n", result.parse_tree);
      }

      pg_query_free_parse_result(result);

      // Optional: clean up the top-level memory context (useful for Valgrind)
      pg_query_exit();

      return 0;
    }
    ```
  </Step>

  <Step title="Compile and run">
    ```bash
    cc -Ilibpg_query -Llibpg_query example.c -lpg_query -o example
    ./example
    ```
  </Step>

  <Step title="Inspect the output">
    You should see the parse tree for `SELECT 1` as JSON:

    ```json
    {
        "version": 170007,
        "stmts": [
            {
                "stmt": {
                    "SelectStmt": {
                        "targetList": [
                            {
                                "ResTarget": {
                                    "val": {
                                        "A_Const": {
                                            "val": {
                                                "Integer": {
                                                    "ival": 1
                                                }
                                            },
                                            "location": 7
                                        }
                                    },
                                    "location": 7
                                }
                            }
                        ],
                        "limitOption": "LIMIT_OPTION_DEFAULT",
                        "op": "SETOP_NONE"
                    }
                }
            }
        ]
    }
    ```

    The `version` field is the PostgreSQL version number (`170007` = PG 17.7). The `stmts` array contains one entry per SQL statement.
  </Step>
</Steps>

## Error handling

Always check the `error` field before using the result. If parsing fails, `error` is non-NULL and contains the error message and cursor position:

```c
PgQueryParseResult result = pg_query_parse("SELECT * FORM users");

if (result.error) {
  printf("error: %s at position %d\n",
         result.error->message,
         result.error->cursorpos);
}

pg_query_free_parse_result(result);
```
